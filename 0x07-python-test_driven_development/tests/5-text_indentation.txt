==================
5-text_indentation
==================

This function that prints a text with 2 new lines after each of these characters: '.', '?' and ':'.

Import the function:
    >>> text_indentation = __import__('5-text_indentation').text_indentation

If argument is a string:
    >>> "Hello. How's ALX treating you? Great?"
    >>> text_identation(text)
    Hello.
    <BLANKLINE>
    How's ALX treating you?
    <BLANKLINE>
    Great?

if argument is not a string:
    >>> text_identation(7)
    Traceback (most recent call last):
    TypeError: text must be a string

Only identation characters:
    >>> text_identation(".?:")
    .
    <BLANKLINE>
    ?
    <BLANKLINE>
    :
    <BLANKLINE>

One special character:
    >>> text_identation("?")
    .
    <BLANKLINE>

None is passed through:
    >>> text_identation(None)
    Traceback (most recnt call last):
    TypeError: text must be a string

An empty string is passed through:
    >>> text_identation("")

Nothing is passed through:
    >>> text_identation()
    Traceback (most recent call last):
    TypeError: text_indentation() missing 1 required positional argument: 'text'
